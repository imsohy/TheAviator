# 안개(Fog) 비활성화

## 목적

멀리 있는 오브젝트가 **안개에 의해 뿌옇게 가려지는 효과**를 없애고, 카메라 far 클리핑 범위 안에서는 거리에 상관없이 색이 밝아지며 사라지지 않게 하기 위함입니다.

## 변경 내용

기존에는 `THREE.Fog(0xf7d9aa, near, far)`로 원근에 따라 배경색으로 수렴하는 안개가 켜져 있었습니다. 아래처럼 **안개를 끈 상태**로 두었습니다.

| 파일 | 설정 |
|------|------|
| `src/game.js` | `scene.fog = null;` |
| `src/part1.js` | `scene.fog = null;` |
| `src/part2.js` | `scene.fog = null;` |

## 참고

- **여전히 보이지 않는 경우**는 카메라 **`far` 평면(메인 게임에서는 10000)** 밖으로 나간 기하만 해당합니다. 안개와는 별개입니다.
- 안개를 다시 켜려면 예: `scene.fog = new THREE.Fog(0xf7d9aa, 100, 950);` 형태로 복원하면 됩니다.
